# Achievement artwork

Badge face illustrations sourced from GitHub's profile achievements, catalogued
at https://github.com/Schweinepriester/github-profile-achievements
(original assets © GitHub / github.githubassets.com).

Downscaled to 256px and pngquant-quantized before committing (installer-size
policy). Mapping to TokenTracker badge ids lives in
`dashboard/src/ui/achievements/badge-catalog.js`.
